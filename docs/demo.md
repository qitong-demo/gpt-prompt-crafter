# GPT Prompt Crafter Demo

下面是一个示例，展示如何用 `gpt-prompt-crafter` 把模糊想法变成可复用提示词。

## 示例输入

```text
Use $gpt-prompt-crafter to turn my rough task idea into a structured prompt.

我想让 GPT 帮我写小红书文案，主题是 AI 工具推荐。
要求自然一点，不要太营销。
```

## 示例输出

### 1. 需求理解

你需要一个用于生成小红书风格内容的提示词，主题是 AI 工具推荐。文案要自然、有真实使用感，避免硬广和夸张营销语。

### 2. 最终提示词

```text
你是一名擅长小红书内容创作的中文文案编辑。

请围绕“AI 工具推荐”生成一篇小红书笔记文案。

要求：
- 语气自然，像真实用户分享使用经验。
- 不要使用夸张营销话术。
- 开头要能引起目标读者兴趣。
- 正文包含工具适合谁、解决什么问题、实际使用感受。
- 结尾给出轻量互动引导。
- 输出 3 个标题备选和 1 篇正文。

输出格式：
1. 标题备选
2. 正文
3. 适合添加的标签
```

### 3. 优化说明

原始需求只说明了主题和风格，优化后补充了角色、平台语境、内容结构、语气限制和输出格式，让模型更容易稳定生成可用文案。

### 4. 可选增强版

```text
你是一名长期运营小红书账号的内容策划，擅长把工具类产品写成真实、克制、有信息量的经验分享。

请基于以下主题生成一篇小红书笔记：
主题：AI 工具推荐

目标读者：
- 学生
- 职场新人
- 内容创作者

写作要求：
- 像真实体验分享，不要像广告。
- 多写具体使用场景，少写空泛优点。
- 避免“神器”“闭眼入”“颠覆效率”等夸张词。
- 语言轻松，但信息密度要够。

输出：
1. 5 个标题
2. 1 篇正文
3. 5 到 8 个标签
4. 可替换开头 2 个
```

## 使用前后对比

使用前：只有“帮我写小红书文案”这类宽泛描述，输出容易不稳定。

使用后：得到可直接复用的完整 Prompt，包含角色、任务、风格、约束和输出格式。
