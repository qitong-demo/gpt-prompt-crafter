# GPT Prompt Crafter 使用说明

`gpt-prompt-crafter` 是一个用于把模糊需求整理成高质量 GPT 提示词的 Codex Skill。

它适合在你只有一个粗略想法、零散要求或口语化描述时使用，帮助你生成结构清晰、可直接复制给模型执行的提示词。

## 适用场景

- 从一句模糊需求生成完整提示词。
- 优化已有提示词，让输出更稳定。
- 编写系统提示词、角色提示词、任务提示词或工作流提示词。
- 补充输出格式、约束、质量标准和边界情况。
- 把中文或英文需求整理成可复用 Prompt 模板。

## 调用方式

在 Codex 中输入：

```text
Use $gpt-prompt-crafter to turn my rough task idea into a structured prompt.
```

然后补充以下信息：

- 你想让模型完成什么任务
- 使用场景和目标用户
- 输入内容大概是什么
- 你希望输出成什么格式
- 风格要求、限制条件或示例
- 目标模型，如果不是 GPT-5.4 可以说明

## 输出结构

Skill 默认使用中文输出，并按以下结构组织结果：

1. 需求理解
2. 最终提示词
3. 优化说明
4. 可选增强版

## 使用建议

如果需求很简单，Skill 会直接给出可用提示词，不会过度追问。

如果缺少关键条件，它只会问最重要的 3 到 5 个问题，避免把提示词优化变成冗长访谈。
